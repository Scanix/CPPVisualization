Selection

The panel on the left let you select the data you want to display. It affects all graphics.

The file tree in the bottom left lets you pick only the files you want to display. You can use CTRL to select multiple files. CTRL click on a folder to select all the items inside it.

Many display options are available:

  • "Automatic file selection" - This can be used to automatically add headers and source files to the graph, if they are related to the selection in the file tree.
  • "Automatic file selection include direction" - This defines in which direction automatically added files are retrieved.
  • "Mouse hover settings" - Defines which file should be highlighted when the mouse is hover. This only applies to the depency tree.
  • "Show external libraries" - Add libraries that don't have a corresponding file to the graph.

Buttons under the "selection" menu allows to automatically select files of a kind below.
